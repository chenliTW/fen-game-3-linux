# 玩了會懷疑人生系列-3-俄羅斯方塊(Linux版)
COMPILED WITH g++ RUN IN DEBAIN

1. 執行下列指令
```
 sudo apt-get install git libncurses5-dev g++ -y
 git clone https://github.com/chenliTW/fen-game-3-linux.git
 cd fen-game-3-linux
 g++ -lcurses -o teris main.cpp fn.cpp
 ./teris
```
